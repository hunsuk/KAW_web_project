# python Flask 서버 실행

1. 클론 후 ../QuizProject/QuizProject_AI 폴더에 있는 .venv를 이용하여 가상환경 실행
2. pip install 을 통해 requermentlib에 있는 라이브러리를 설치 할 것
3. python webServer.py를 입력 하여 서버 실행
